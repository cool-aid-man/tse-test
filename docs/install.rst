.. title:: AdvancedTagScript installation options


==========
Installing
==========


pip
^^^

`package <https://pip.pypa.io/>`_::

    $ pip install AdvancedTagScript

Latest version of `AdvancedTagScript` supports only python3.


Source
^^^^^^

Download `source <https://pypi.org/project/AdvancedTagScript/>`_::

    $ pip install -e .


Git Repository
^^^^^^^^^^^^^^

Get latest development version::

    $ git clone https://github.com/cool-aid-man/TagScriptEngine.git
